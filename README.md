# 🦠 COVID-19 Global Data Analysis

## 📌 Project Overview

The COVID-19 pandemic affected millions of people worldwide and generated massive amounts of data. In this project, I analyzed global COVID-19 data to uncover trends in confirmed cases, deaths, and vaccination progress across different countries.

Using Python and SQL, I cleaned, explored, and visualized real-world data to transform raw numbers into meaningful insights.

---

## 🎯 Project Goals

* Understand the global impact of COVID-19.
* Analyze infection and death trends over time.
* Compare COVID-19 statistics across countries.
* Explore vaccination progress worldwide.
* Build practical data analysis skills using real-world datasets.

---

## 📂 Dataset

**Source:** Our World in Data COVID-19 Dataset

The dataset contains:

* Country-wise COVID-19 cases
* Daily reported cases
* Total deaths
* Population statistics
* Vaccination data
* Historical records by date

---

## 🧹 Data Cleaning

Before analysis, the data was cleaned and prepared by:

✔ Handling missing values

✔ Converting date columns into proper datetime format

✔ Removing unnecessary records

✔ Preparing data for analysis and visualization

## 🔍 Analysis Performed

### 📈 Trend Analysis

* Growth of COVID-19 cases over time
* Daily and cumulative case analysis

### 🌍 Country Comparison

* Most affected countries
* Highest death counts
* Infection rate comparisons

### 💉 Vaccination Analysis

* Vaccination progress across countries
* Relationship between vaccinations and pandemic trends

---

## 📊 Key Insights

🔹 Several countries experienced multiple COVID-19 waves during the pandemic.

🔹 Case growth patterns differed significantly across regions.

🔹 Vaccination rates increased rapidly after vaccine rollout programs.

🔹 Data visualization revealed important trends that are difficult to identify from raw data alone.
